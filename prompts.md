# 🎧 Prompt Pack for 1001 Albums MCP

These prompts demonstrate what AI assistants can do with the MCP server.

---

# Daily Listening

```
What's today's album from my 1001 albums project?
```

```
Give me background about today's album and why it's historically important.
```

```
Write a short review of today's album in the style of Pitchfork.
```

---

# Listening History

```
Show me the last 10 albums I've listened to.
```

```
Which artists appear most often in my listening history?
```

```
Which decade dominates my listening history?
```

---

# Taste Analysis

```
Based on my ratings, what genres do I like the most?
```

```
Which albums did I rate much higher than the community average?
```

```
Which albums did I rate much lower than the community?
```

---

# Discovery

```
Find connections between the last five albums I listened to.
```

```
Explain the musical lineage connecting these albums.
```

```
Identify recurring producers or collaborators in my listening history.
```
